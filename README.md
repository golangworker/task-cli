# Трекер задач CLI

### Требоваяния

- [ ] 1. Возможность добавления и удаления задач

- [ ] 2. Уметь отмечать задачу как выполненную

- [ ] 3. Выводить список всех задач

- [ ] 4. Выводить список всех выполненных задач

- [ ] 5. Выводить список всех невыполненных задач

- [ ] 6. Выводить список задач, которые в процессе выполнения

- [ ] 7. Использовать JSON-файл для хранения задач в текущем каталоге (если JSON-файл не существует, его нужно создать)е случаи

- [ ] 8. Корректно обрабатывать ошибки и пограничны
#### Структура Task
- `id`: Уникальный идентификатор задачи
    
- `description`: Краткое описание задачи
    
- `status`: Статус задачи ( `todo`, `in-progress`, `done`)
    
- `createdAt`: Дата и время создания задачи.
    
- `updatedAt`: Дата и время последнего обновления задачи.
